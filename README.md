This task, we analyze data using logical and agregate function. 

`AND Function` the condition is
```
  IF A2>B2 AND A2>C2 = TRUE
    RETURN "Apples"
  ELSE
    RETURN "No apples"
```
`OR Function` the condition is
```
  IF A3>B3 OR A3>C3 = TRUE
    RETURN "Yes"
  ELSE
    RETURN "No"
```
`NOT Function` the condition is
```
  NOT (TRUE)
    RETURN "Yes"
```
